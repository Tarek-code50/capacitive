# تقرير تفاعلي: سعة خطوط النقل

صفحة ويب تفاعلية توضح تأثير السعة في خطوط النقل الكهربائية مع رسوميات تفاعلية وحاسبة تيار الشحن.

كيفية الاستخدام:

- افتح `index.html` في متصفح حديث (Chrome, Edge, Firefox).
- العناصر التفاعلية تعتمد على اتصال إنترنت للوصول إلى CDN (Tailwind, Chart.js, Plotly).

نشر على GitHub:

1. أنشئ مستودعًا جديدًا في GitHub.
2. انسخ الملفات من هذا المجلد (`index.html`, `README.md`, وغيرها) إلى المستودع.
3. ادفع التغييرات إلى الفرع `main`.

اختبار محلي (PowerShell):

```powershell
# إعادة تسمية الملف (إن لزم)
Rename-Item -Path "Untitled-1.ini" -NewName "index.html"
# أو افتح الملف مباشرة
Start-Process "index.html"
```

ملاحظات:

- الملف الأصلي كان بلاحقة `.ini`، وتم إنشاء `index.html` جاهز للعرض.
- إن واجهت أخطاء في الكونسول عند العرض، أرسل لي لعرض سجل الأخطاء وسأصحّحها.
